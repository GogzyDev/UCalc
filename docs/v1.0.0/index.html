UCalc

UCalc is a lightweight and efficient expression parser designed specifically for Unity.
It allows you to serialize lambda expressions directly in the Unity Inspector, making it easy to expose customizable logic and calculations without writing additional scripts.

UCalc integrates seamlessly with the Unity Editor and provides a powerful yet intuitive way to define expressions that are compiled into fast runtime delegates.

Why UCalc?

UCalc makes it possible to move small pieces of logic into data by allowing expressions to be defined directly in serialized fields.

Common use cases include:

  • Configurable formulas
    Example: damage calculations, score multipliers, stat scaling.

  • Custom gameplay logic
    Designers can define simple rules without requiring new code.

  • Advanced persistent events
    Go beyond what UnityEvent offers by allowing serialized lambda listeners.

  • Data-driven systems
    Expressions can reference context objects, arguments, and dependencies.

Unity Editor Integration

UCalc is built with Unity workflows in mind.

Features include:

  • Serialized lambda expressions
  • Expression validation
  • Autocomplete support
  • Reflection-based symbol discovery
  • Integration with Unity objects and components

Expressions are compiled into fast delegates using expression trees, ensuring runtime performance remains excellent.

Example

A serialized lambda can expose arguments and be written directly in the Inspector:

health * damageMultiplier + bonusDamage

At runtime this expression is compiled into a delegate equivalent to:

(float health, float damageMultiplier, float bonusDamage) => health * damageMultiplier + bonusDamage;
